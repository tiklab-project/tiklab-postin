用户管理 UserController
POST /user/createUser 创建用户
POST /user/findUser 根据用户ID查找用户
POST /user/updateUser 更新用户
POST /user/deleteUser 根据用户ID删除用户
POST /user/findAllUser 查找所有用户

组织管理 OrgaController
POST /orga/findOrga 根据ID查找组织
POST /orga/createOrga 创建组织
POST /orga/updateOrga 更新组织
POST /orga/findAllOrga 查找所有组织列表
POST /orga/deleteOrga 删除组织
